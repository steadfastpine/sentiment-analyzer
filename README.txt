Sentiment Analyzer



# Contact: https://www.linkedin.com/in/steadfastpine/

# Release Date: 2019-06-10
# Version: .4



Description

	Counts the occurrence of positive and negative words within a body of text.
	Provides a report indicating if the sentiment of the text is either positive or negative.


Prerequisites

	Python 3



Installation

	Download and unzip the program files, then change working directory to them:
	
		# cd sentiment-analyzer


	Execute the program with your favorite interperetor such as idle, or from the command line:

		# python sentiment_analyzer.py
